# Code of Conduct

Hermes Fleet Control welcomes contributors who want to make local-first AI-agent operations safer and easier to run.

## Expected behavior

- Be respectful and specific.
- Assume good intent, but verify claims with evidence.
- Keep discussions focused on the project and user safety.
- Give actionable feedback in issues and pull requests.
- Respect privacy; do not ask contributors to share secrets or private local state.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Publishing another person's private information.
- Pressuring users to share tokens, credentials, private IDs, logs, or connection strings.
- Repeatedly derailing technical discussions.
- Submitting code that intentionally weakens safety gates or redaction without clear review.

## Reporting

If you see behavior that violates this code, open a maintainer-facing issue or use the repository's private contact path if available. Keep reports factual and avoid including private secrets or raw personal data.

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code.
